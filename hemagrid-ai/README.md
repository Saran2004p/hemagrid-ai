# 🩸 HemaGrid AI — Blood Warriors Foundation

> Connecting Donors. Saving Lives.

## 🚀 Quick Start

```bash
# Step 1: Install dependencies
npm install

# Step 2: Run locally
npm run dev
# Opens at http://localhost:5173

# Step 3: Build for deployment
npm run build
```

## 📁 Project Structure
- `src/components/` — Reusable UI components
- `src/pages/`      — 6 page views (Home, Donors, Patients, How It Works, About, Contact)
- `src/locales/`    — 11 language files (English + 10 Indian languages)
- `public/logo.png` — HemaGrid AI logo

## 🌍 Supported Languages
English, Hindi, Tamil, Telugu, Bengali, Marathi, Gujarati, Kannada, Malayalam, Punjabi, Odia

## 🚢 Deploy
```bash
npm run build
# Upload the `dist/` folder to Vercel, Netlify, or any static host
```
